常见问题

  • 不登录能用吗？ 打卡、摸鱼计时、提醒、每日毒鸡汤等工具游客即可使用；AI 对线可游客试用若干次；排行榜、AI 周报、毒舌推送等需登录后使用。
  • 收不到推送怎么办？ 请在 系统设置 → 通知 中允许「狗啊喂」发送通知，并在应用内「我的 → 毒舌推送」开启开关。
  • 毒舌推送太频繁？ 在「我的 → 毒舌推送」可调整强度档（低频/正常/狂轰）或设置免打扰时段，也可一键关闭。
  • 如何注销账号？ 「我的 → 设置 → 注销账号」，账号数据将被永久删除且不可恢复。
  • 对线/周报内容如何生成？ 由第三方大模型基于你的输入与职场画像生成，详见 隐私政策。
