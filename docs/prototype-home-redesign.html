HomeScreen 改版 · v2 · 核心动线重构

核心洞察：大部分用户打开 App 就是不知道吃什么 → 「来点灵感」是主路径，不能边缘化

核心动线设计原则
用户分叉点
有想法 → 文字 / 语音输入 → 开始推荐
没想法 → 来点灵感 → 直接进入推荐结果
两条路径平权，灵感路径视觉上更显眼（因为这是更高频的场景）
改动前的问题
❌ 来点灵感被缩成右上角小胶囊 ❌ 输入区做主角但新用户没有输入方向 ❌ 两条路径权重失衡
新布局顺序
① Header — App 图标（真实）+ 今天吃点啥？
② 来点灵感卡片 — 暖橙渐变大卡，品牌色做底，醒目 CTA
③ 分隔符 — "或者有方向了？"
④ 输入卡片 — 文字/语音 Tab + 显式提交按钮
来点灵感卡片
· 全宽暖橙渐变背景（品牌主色），建立视觉锚点
· 展示关键数字：300 道家常菜、荤素搭配、热量合理
· 一个「来点灵感，出发！」大按钮，点击直接跳结果
· 点阵纹理背景增加质感，避免纯色单调
输入卡片
· Tab 切换（📝 文字 / 🎙 语音）取代隐藏的 mode-toggle
· 文字模式：输入框 + 显式「开始推荐 →」按钮
· 语音模式：大圆形麦克风（tap 即录）+ 转写后显示按钮
· 空内容时按钮置灰，防止空提交
App 图标
使用真实 App 图标图片（1024×1024 圆角），不用 emoji 代替。
Header 中 46×46px，圆角 13px，与 iOS 图标视觉一致。

HomeScreen.js — 布局重构（主改动）
